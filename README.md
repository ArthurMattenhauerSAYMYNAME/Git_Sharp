## Projeto escolar de uma Calculadora em C#

Projeto de uma calculadora em Windows Forms (C#), onde apresenta as seguintes funções:
- Somar dois números
- Subtrair dois números
- Multiplicar dois números
- Dividir dois números
- Comparar dois números (maior, menor ou igual)
- Par ou ímpar (fala se os números são pares ou impares)
    - Botão de limpar (limpa o sinal, resultado e números colocados)
 
Também foram usados os seguintes componentes do Windows Forms:
1. Text Box
2. Label
3. Button
